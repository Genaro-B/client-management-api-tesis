# Mejoras del Bot de Telegram

## Objetivo

Documentar ideas, mejoras y referencias para evolucionar el flujo conversacional del bot de Telegram antes de integrar IA.

Este documento es un espacio de trabajo. Las ideas aquí pueden cambiar, descartarse o implementarse más adelante.

---

# Objetivos principales

- Mejorar la experiencia del usuario.
- Hacer la conversación más natural.
- Reducir la cantidad de texto que el usuario debe escribir.
- Guiar al usuario mediante botones y menús.
- Mantener un flujo simple y fácil de extender.

## Mejoras UX

Ideas para evaluar:

- Mensajes más naturales.
- Confirmaciones antes de finalizar.
- Evitar respuestas ambiguas.
- Manejo de errores.
- Respuestas para comandos desconocidos.
- Botón "Cancelar".
- Botón "Volver".
