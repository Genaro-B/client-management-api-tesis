# API Tesis

Sistema de automatización para la gestión de clientes mediante integración de servicios e inteligencia artificial.

## Descripción

Este proyecto implementa una API REST para la administración de clientes.

La solución forma parte del trabajo final de investigación titulado:

**"Automatización de Procesos basada en Integración de Servicios e Inteligencia Artificial en Entornos de Bajo Costo"**

La API será consumida por flujos de automatización desarrollados en n8n y por interfaces conversacionales basadas en Telegram.

---

## Objetivos

* Centralizar la gestión de clientes.
* Proporcionar una API REST documentada.
* Facilitar la integración con herramientas de automatización.
* Servir como componente central de una arquitectura orientada a servicios.

---
